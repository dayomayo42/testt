# ProfApp iOS

Приложение ProfApp для платформы iOS.
https://apps.apple.com/ru/app/profapp-запись-клиентов-и-crm/id1553889063

## Описание проекта

ProfApp - приложение-органайзер для работы специалистов различных сфер, работающих с клиентами. Приложение содержит календарь-расписание, калькулятор прибыли/убыли, онлайн запись, новостной блок, склад для учета товаров и тд для упрощение работы специалистов. На первые 14 дней с момента регистрации дается пробная подписка. 

## Настройка и запуск

Клонировать репозиторий или скачать проект в виде архива. В терминале переходим в папку проекта и выполняем команду `pod install` для установки зависимостей. Для обновления зависимостей можно выполнить команду `pod update`. Запускаем файл workspace проекта (*iProfi_new.xcworkspace*).

## Структура проекта

```
- Iprofi_new.xcodeproj
    iProfi_new
    OneSignalNotificationServiceExtension
    Products
    Pods
    Frameworks
```
В папке *iProfi_new* находится основной код проекта

В папке *OneSignalNotificationServiceExtension* находится расширение для работы с уведомлениями от One Signal

Рассмотрим структуру папки iProfi_new:

```
- iProfi_new
    Modules
    Font
    View
    Networking
    Extension
```
В папке *Modules* находятся модули экранов приложения

В папке *Font* находятся кастомные шрифты приложения

в папке *View* находятся отдельные переспользуемые экраны приложения

в папке *Networking* находится модуль сетевого слоя, написанный на Moya и RX

в папке *Extension* находятся расширения для базовых классов приложения

## Code style

Со спецификацией code style можно ознакомится по [ссылке](https://git.midev.pro/profapp-ios/profapp_ios/-/blob/main/CodeStyle.md).

Весь новый код в приложении должен по возможности соответствовать данному стилю.
Legacy код приводится в соответствие, если возникает необходимость его рефакторинга или изменения.
